# 出差发票自动整理系统

自动化处理百度网盘中的出差发票文件，通过 OCR 识别发票信息并规范化命名，按类别存储到本地目录。

## 功能特点

- **自动同步**：通过 bypy 从百度网盘自动下载新发票
- **智能识别**：使用 PaddleOCR 免费识别发票内容
- **自动分类**：支持机票、火车、打车、住宿、餐饮等多种发票类型
- **规范命名**：按日期、类型、金额、出差人等信息自动重命名
- **分类存储**：按年月和发票类型自动组织目录结构

## 系统架构

```
百度网盘云端 → bypy下载 → 临时目录 → OCR识别 → 解析信息 → 重命名 → 本地整理目录
```

## 发票类型支持

| 类型 | 命名格式 | 示例 |
|-----|---------|------|
| 机票 | `{日期}_机票_{出发地}_{目的地}_{金额}_{出差人}.pdf` | `2025-02-20_机票_北京_上海_850.00_张三.pdf` |
| 火车 | `{日期}_火车_{出发地}_{目的地}_{金额}_{出差人}.pdf` | `2025-02-20_火车_南京_杭州_125.50_张三.pdf` |
| 打车 | `{日期}_打车_{出发地}_{目的地}_{金额}_{出差人}.pdf` | `2025-02-20_打车_上海站_希尔顿酒店_45.00_张三.pdf` |
| 住宿 | `{日期}_住宿_{城市}_{酒店名}_{金额}_{出差人}.pdf` | `2025-02-21_住宿_上海_希尔顿酒店_480.00_张三.pdf` |
| 餐饮 | `{日期}_餐饮_{城市}_{金额}_{出差人}.pdf` | `2025-02-20_餐饮_上海_128.00_张三.pdf` |
| 租车 | `{日期}_租车_{城市}_{天数}天_{金额}_{出差人}.pdf` | `2025-02-20_租车_深圳_3天_450.00_张三.pdf` |

## 安装

### 1. 克隆项目

```bash
cd /path/to/travel
```

### 2. 安装依赖

```bash
pip install -r requirements.txt
```

### 3. 配置 bypy

首次使用需要授权百度网盘：

```bash
bypy info
```

按照提示在浏览器中完成授权。

### 4. 修改配置

编辑 `config/config.yaml`，设置：
- 默认出差人
- 本地输出目录
- 处理选项

编辑 `config/travelers.yaml`，添加出差人信息。

## 使用

### 执行一次任务

```bash
python main.py --run
```

### 启动每日定时任务

```bash
python main.py --daily
```

### 模拟运行（不实际移动文件）

```bash
python main.py --run --dry-run
```

## 配置说明

### config.yaml - 主配置

```yaml
baidu_pan:
  remote_dir: "invoices"     # 百度网盘远程目录
  temp_dir: "temp"           # 本地临时目录

local_output_dir: "invoices" # 整理后输出目录

default_traveler: "张三"      # 默认出差人

options:
  delete_temp_after_process: true  # 处理后删除临时文件
  skip_existing: true              # 跳过已处理文件
  dry_run: false                   # 模拟运行
```

### parsers.yaml - 发票解析规则

定义各类发票的识别规则和命名模板。

### travelers.yaml - 出差人信息

定义出差人列表及其别名。
